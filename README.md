# GuqinAdoBP
> 古琴领养计划BP

## 项目名称
古琴领养 - 免费带张琴回家

### 项目概述

中国传统文化里，群星璀璨，其中最耀眼一颗，莫过于四艺之首的古琴。

古琴能修己按人，精神如星河深邃，意蕴悠远。

然而古琴工艺复杂，材质金贵，对普通人，地位与价格均可望不可即。

古琴领养计划为解决这个问题而生。

只用押金4500元/15000元，而不用购买，更没有租金。免费使用一年半期限，到期爱上了就带走，不爱琴金两相还。


- 团队项目协作仓库地址 [领养张琴回家吧](https://github.com/nanshanqinshe/GuqinAdoBP)  



真实项目。

- 8w、9w 集中打磨 BP
- 10w 路演

- 问题
如何解决古琴高认知负荷和高价格两个问题。
